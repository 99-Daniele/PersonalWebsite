Progetto Tesi Magistrale
Set 2023 - Lug 2024

Il mio progetto di tesi magistrale, intitolata"Analysis of the Trials of the Civil Court of Milan" rigurdava l'analisi dati dei procesimenti giudizioari del Tribunale Civile di Milano. L'obiettivo era di identificare le cause delle lunghe durate dei procedimenti.

I dati consisteavano in un database contenente tutti gli eventi giudici registrati, i quali ho pulito, aggregato e trasformato in informazioni utili riguardanti la struttura e la durata dei procesimenti. Ho quindi creato un software da zero, implementato in Python, che permette all'utente di interagire, filtrare e visualizzare grafici.

Inoltre, siccome molti dei procedimenti analizzati erano ancora in corso, ho implementato un modello di regressione lineare per predirre la durata finale di questo tipo di procedmenti. Ingine, ho analizzato i risultati per identificare le criticità e i èrpbòemi relativi al sistema giudiziario civile.

Java / UML / Testing / Lavoro di Gruppo / Management del Progetto

Progetto Ingegneria del Software
Mar 2021 - Lug 2021

Per la mia tesi triennale, ho partecipato a un progetto di gruppo focalizzato sull'Ingegneria del Software, dove abbiamo sviluppato un'applicazione Java.

L'obiettivo era di implementare il pattern Model-View-Controller (MVC) utililzzando i paradigmi di Programmazione Object-Oriented.

Il software simulava un gioco da tavolo utilizzando un'architettura Client-Server, permettendo ai giocatori di creare partite multiple in contemporanea.

Un aspetto cruciale del progetto riguardano i test di validation e integrazioni per identificare bugs e potenziali problemi.

Java / HTML / CSS / Javascript / MySQL / Apache Tomcat

Progetto Sviluppo Web
Mar 2021 - Lug 2021

Questo progetto consiste nel creare un'applicazione web da zero, implementando un sito web di e-commerce.

La fase iniziale di focalizzava sul creare il diagramma Entità-Relazioni (ER), implementarlo in un database MySWL, e allestire un server Apache Tomcat su cui ospitare l'applicazione.

Successivamente, ho implementato il back-end usando Java e il front-end usando HTML e CSS.

Inoltre, l'implementazione includeva due versioni: una standard HTML e una Rich Internet Application (RIA) che utilizza Javascript per migliorare l'esperienza utente.

HTML / Vue / Nuxt.js / Vercel / Github Pages / Lavoro di Gruppo

Progetto Sviluppo Front-End
Mar 2023 - Lug 2023

Questo progetto si divide in due parti principali: analizzare e valutare il design e la user experience di un sito web esistente, e implementare il front-end di un nuovo sito web in base a ciò che avevamo imparato.

Ogni componente del gruppo ha dato una valutazione individuale, seguita dalla valutazione collettiva usando criteri e sistemi di valutazione predefiniti.

Per ottenere il feedback di un'esperienza utente generica, abbiamo condotto delle interviste di persona con utenti reali, analizzado il loro comportamento e raccogliendo le loro impressioni.

La seconda parte del progetto riguarda la crezione di un front-end di un fittizio sito web di venture capital, applicando i concetti precedentemente analizzati.
